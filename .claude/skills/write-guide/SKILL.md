このブログ（zipishi-blog）の記事の書き方ガイドを提示してください。以下の内容を分かりやすく説明してください。

## 1. 記事ファイルの構成

記事は `content/posts/` に Markdown ファイルとして配置します。
ファイルの先頭が **Front Matter**（記事情報）、その下が **本文**です。

```
---
（Front Matter: タイトルやタグなどの情報）
---

（本文: マークダウンで書く）
```

## 2. Front Matter の各フィールド

| フィールド | 説明 | 例 |
|---|---|---|
| `title` | 記事タイトル | `"夏の旅行記"` |
| `cover.image` | 一覧に表示されるカバー画像のパス | `"/images/posts/260308-trip/eyecatch.jpg"` |
| `cover.alt` | カバー画像の説明（アクセシビリティ） | `"夏の旅行記のアイキャッチ"` |
| `date` | 公開日時（JST） | `2026-03-08T21:30:00+09:00` |
| `lastmod` | 最終更新日時 | `2026-03-08T21:30:00+09:00` |
| `tags` | タグ（複数可） | `["育児", "旅行", "1歳"]` |
| `categories` | カテゴリー | `["日記"]` |
| `Author` | 筆者 | `"ぢぴ氏"` |
| `draft` | `false` で公開、`true` で下書き | `false` |

## 3. マークダウン記法

### 見出し
```markdown
## 大見出し
### 中見出し
#### 小見出し
```

### 太文字
```markdown
**太文字にしたいテキスト**
```

### 区切り線
```markdown
---
```

### 改行・段落
- テキストの後に半角スペース2つ → 改行
- 空行を1行入れる → 段落（おすすめ）

### 画像の埋め込み
```markdown
![画像の説明](/images/posts/YYMMDD-スラッグ/image.jpg)
```
画像ファイルは `static/images/posts/YYMMDD-スラッグ/` に置く。

### リンク
```markdown
<!-- 外部リンク -->
[リンクのテキスト](https://example.com)

<!-- ブログ内の別記事へのリンク -->
[記事タイトル](/posts/記事スラッグ/)
```

## 4. カスタムショートコード

### 年齢表示（`age`）
ちゃ太郎の年齢を自動計算して表示する。

```
{{</* age birth="2024-09-03" */>}}
```
→ 「0歳11ヶ月」のように表示される

### X（旧Twitter）埋め込み
```
{{</* tweet 1234567890123456789 */>}}
```
ポストのID（URLの末尾の数字）を指定する。

### Instagram 埋め込み
```
{{</* instagram B7VI0ICpSDz */>}}
```
投稿のショートコード（URLの `instagram.com/p/` の後の部分）を指定する。

## 5. 書き方のヒント

- 顔文字やキャラクター性を入れると読みやすくなる（例: `(^^)` `笑`）
- 画像は積極的に入れる（文章だけより断然見やすい）
- リンクも積極的に貼る（内部リンクで他の記事も読んでもらえる）
- `##` より `###` の見出しを使うほうがデザイン的にまとまる
- 書き途中でも `draft: true` にすれば公開されないので安心
